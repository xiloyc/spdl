# Adding a Placement
To add a placement, navigate to the **data** folder and copy **_template.json**. Name the file the level name. (or anything you want honestly.)<br>After that, edit the file you copied and put the information in the json file.<br>After after that (the last one I promise), edit **_list.json** and put the file's name in the placement you'd like.<br>Here is an example:
```
If your level, for example is Top 3, it should be the third one. Like this:

[
   "Top1",
   "Top2",
   "Top3" // It should be this one
]
```
After you have done that, it should appear on the list! If it doesn't, try waiting a bit or clearing your cache.
